# Social Media Engagement Data Analysis

This project performs descriptive analytics on a comprehensive social media engagement dataset to extract actionable insights regarding user interactions, brand performance, and campaign reach.

## Project Objectives
- Process and analyze a dataset of 12,000 social media posts  
- Evaluate likes, shares, comments, and engagement rates  
- Identify top-performing posts and campaigns  
- Understand relationships between different engagement metrics  

## Tools Used
- Python  
- Pandas  
- Matplotlib  

## Analysis Performed
- Data cleaning and preprocessing of a 28-column dataset  
- Creation of a custom Total Engagement metric  
- Statistical summary (mean, standard deviation, quartiles)  
- Identification of top-performing posts based on engagement  
- Correlation analysis between likes, shares, and comments  
- Data visualization using histograms, bar charts, and correlation matrices  

## Dataset Characteristics
- Includes 12,000+ posts across platforms like Instagram, YouTube, Facebook, Twitter, and Reddit  
- Contains timestamps and day-wise posting data  
- Includes content features such as hashtags, mentions, keywords, and language  
- Provides sentiment analysis including sentiment score, toxicity, and emotion  
- Tracks brand-related information like brand name, product, and campaign  

## Insights
- High engagement posts are strongly influenced by shares and comments  
- Engagement varies significantly across different platforms  
- Certain brands consistently achieve higher interaction rates  
- Strong correlation exists between likes, shares, and comments
